##使用说明
* 复制template文件夹到`xxx.farbox.com`文件夹下
* 请在`base.html`中修改大标题，默认为 **C'est  quoi le patron de conception** 需要修改的自行修改。
* 在`configs.html`中可以设置首页出现的文章数目,导航栏的数目对应文章数目。
